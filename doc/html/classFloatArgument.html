Detailed Description

La classe FloatArgument hérite de la classe Argument. Les objets de cette classe stockent les informations d'un argument de type float, elle posséde une variable qui contient sa valeur (m_value). Elle dispose d'un constructeur et des méthodes qui permettent de tester et de définir la valeur d'une instance (testValue()), d'afficher les informations liées à une instance (display()), d'un accesseur (getValue()) et d'un mutateur (setValue(float)).
